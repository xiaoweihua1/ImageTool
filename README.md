## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🗜️ 批量压缩 | 可调质量级别，保留EXIF |
| 🔄 格式转换 | JPG ↔ PNG ↔ BMP ↔ WebP ↔ TIFF |
| 📐 尺寸调整 | 按比例/像素统一缩放 |
| 📂 批量处理 | 整个文件夹拖入处理 |

## 🚀 快速开始

```bash
git clone https://github.com/xiaoweihua1/ImageTool.git
pip install Pillow
python main.py
```
